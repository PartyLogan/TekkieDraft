Tekkie Draft Mode

Set the timers, use the Start/Stop button to control the timers.
Reserve time happens automatically. Use the random button if a team runs out of time.

Click and drag the characters to the slot you want them in. They will swap positions if there is already a character there.

Dota timers: Pick/Ban 35secs, Reserve time 130secs.


Dota pick/ban order.
Reverse the order if Team B going first.
1st Ban Phase: A, B, A, B, A, B
1st Pick Phase: A, B, B, A
2nd Ban Phase: A, B, A, B
2nd Pick Phase: B, A, B, A
3rd Ban Phase: B, A
3rd Pick Phase: A, B
